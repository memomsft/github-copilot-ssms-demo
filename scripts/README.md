# 📁 Scripts - Soluciones de Referencia

> Estos scripts **no son el punto de partida del ejercicio**, son la solución de referencia.

## ¿Cómo usarlos?

Cada lab te pide generar un script con Copilot en lenguaje natural. Una vez generado, compara contra el script de referencia de esta carpeta:

| Script | Lab | Qué comparar |
|--------|-----|--------------|
| `diagnostico_bloqueos.sql` | Lab 4 — Ejercicio 4.1 | ¿Usó las mismas DMVs? ¿Incluyó el texto de la query bloqueante? |
| `performance_queries.sql` | Lab 4 — Ejercicio 4.2 | ¿Cubrió query stats, índices sin uso y wait stats? |
| `backup_automatizado.sql` | Lab 4 — Ejercicio 4.3 | ¿Generó ambos jobs (Full + Log)? ¿Con compresión y checksum? |

## ¿Por qué puede diferir lo que Copilot genera?

Copilot es generativo, no produce siempre el mismo output. Diferencias comunes: alias distintos, columnas extra, estilo de formato. Son normales y parte del aprendizaje.
